# Fashion MNIST Classification using CNN

This project implements a Convolutional Neural Network (CNN) for classifying fashion images from the Fashion MNIST dataset using TensorFlow and Keras.

---

# Project Overview

Fashion MNIST is a grayscale image dataset containing different categories of clothing items. The objective of this project is to train a CNN model that can accurately identify and classify fashion products.

The project demonstrates the complete deep learning workflow including:

* Data preprocessing
* Image reshaping
* CNN model creation
* Training and evaluation
* Prediction on test images

---

# Dataset Information

The Fashion MNIST dataset contains:

* 60,000 training images
* 10,000 testing images
* 28 × 28 grayscale images
* 10 different fashion categories

## Classes

| Label | Category    |
| ----- | ----------- |
| 0     | T-shirt/top |
| 1     | Trouser     |
| 2     | Pullover    |

| 4     | Coat        |
| 5     | Sandal      |
| 6     | Shirt       |
| 7     | Sneaker     |
| 8     | Bag         |
| 9     | Ankle boot  |

# CNN Architecture

The model includes:

* Convolution Layers
* ReLU Activation Function
* Max Pooling Layers
* Flatten Layer
* Dense Layers
* Softmax Output Layer

---

# Model Performance

* Training Accuracy: 89.72%
* Testing Accuracy: 88.08%

The model achieved good classification performance on the Fashion MNIST dataset.
